Court Assignment (Interval Allocation)

Problem Description

Given a list of time intervals representing bookings, assign each interval to a court such that no two overlapping intervals are placed on the same court. You can use an infinite number of courts.

Function Signature
  • Map<Integer, List<int[]>> assignCourts(List<int[]> intervals)
Example
Input: [[1, 4], [4, 5], [2, 4]]
Output:
{
  1: [[1, 4], [4, 5]],
  2: [[2, 4]]
}
Notes
  • Intervals are given as [start, end] where start < end.
  • Two intervals overlap if they intersect in time: !(end1 <= start2 || end2 <= start1).
  • The goal is to assign each interval to a court with no overlaps on the same court.
  • You may assume the input is valid and contains no duplicates.
  • One possible strategy is similar to solving "Meeting Rooms II" using a priority queue.

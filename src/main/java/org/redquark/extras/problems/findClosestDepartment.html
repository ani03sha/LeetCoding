Find Closest Department

Difficulty: Medium

Problem Statement

You're given a hierarchical structure of a company. It consists of:

  • A root node representing the Company
  • Each Department belongs to the company or another department
  • Each Employee belongs to exactly one department

Given a list of employees, find the closest common department that contains all of them (directly or via their ancestors).

Input

  • Map<String, List<String>> orgChart — hierarchy from parent to children
  • Map<String, String> employeeToDepartment — map of employee to department
  • List<String> employees — employees to analyze

Example

Input:

orgChart = {
  "Company": ["Engineering", "Sales"],
  "Engineering": ["Platform", "Infra"],
  "Platform": ["Frontend", "Backend"]
}
employeeToDepartment = {
  "Alice": "Frontend",
  "Bob": "Backend",
  "Carol": "Infra"
}
employees = ["Alice", "Bob"]

Output:

"Platform"

Constraints

  • Hierarchy is a valid tree rooted at "Company"
  • No cycles
  • All employees exist in employeeToDepartment

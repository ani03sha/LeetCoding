Design Most Popular Tracker

Description

Design a system that keeps track of content popularity and allows querying the most popular content item at any point in time.

You must implement the following interface:

interface MostPopular {
    void increasePopularity(Integer contentId);
    void decreasePopularity(Integer contentId);
    Integer mostPopular();
}

Rules

  • increasePopularity(contentId) increases the popularity of the content by 1.
  • decreasePopularity(contentId) decreases the popularity by 1 (but never below 0).
  • mostPopular() returns the contentId with the highest popularity.
  • In case of a tie, return any one of the tied contentIds.

Constraints

  • All content IDs are integers in the range 1 to 106.
  • Popularity can grow without bound but must be â‰¥ 0.

Follow-up

Implement a method that returns the most recent contentId whose popularity changed and which was at some point the most popular.

Example

Input:


popularityTracker.increasePopularity(7);
popularityTracker.increasePopularity(7);
popularityTracker.increasePopularity(8);
popularityTracker.mostPopular();        // returns 7
popularityTracker.increasePopularity(8);
popularityTracker.increasePopularity(8);
popularityTracker.mostPopular();        // returns 8
popularityTracker.decreasePopularity(8);
popularityTracker.decreasePopularity(8);
popularityTracker.mostPopular();        // returns 7
popularityTracker.decreasePopularity(7);
popularityTracker.decreasePopularity(7);
popularityTracker.decreasePopularity(8);
popularityTracker.mostPopular();        // returns -1

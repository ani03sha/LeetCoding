Merge Trees with Sum Values

Medium
You are given two trees where each node contains a key and a value. The value of each node represents the sum of its own value plus all its children's values. When merging two trees, you need to:

• Create absent branches in the result tree
• Sum values for existing branches
• Maintain the property that keys are unique among child nodes of a single parent
• Child nodes can be stored in any order

Example 1:

Input Trees:
Tree 1:
Root:10 / | \ A:2 B:2 M:6 | | L:2 L:2
Tree 2:
Root:13 / | \ A:4 B:3 D:6 / \ L:4 M:6
Output:
Root:23 / | \ \ A:6 B:5 D:6 M:6 / | \ L:6 L:2 M:6
Explanation:
• Root nodes: 10 + 13 = 23
• Node A: 2 + 4 = 6, with child L: 2 + 4 = 6
• Node B: 2 + 3 = 5, with child L: 2 + 0 = 2 (only exists in Tree 1)
• Node D: 0 + 6 = 6 (only exists in Tree 2), with child M: 0 + 6 = 6
• Node M: 6 + 0 = 6 (only exists in Tree 1)

Method Signature:

public TreeNode mergeTrees(TreeNode tree1, TreeNode tree2) { // Your implementation here }

Constraints:

  • The number of nodes in both trees is in the range [0, 2000]
  • Node keys are non-empty strings with length ≤ 10
  • Node values are in the range [-1000, 1000]
  • Keys are unique among siblings (children of the same parent)
  • Trees can have different structures

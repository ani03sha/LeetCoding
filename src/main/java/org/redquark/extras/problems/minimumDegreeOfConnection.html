Extra 19. Minimum Degree of Connection

Medium Graph BFS Bidirectional Search Shortest Path
You are given a social network represented as an undirected graph where users are connected through direct friendships. For any two given users S and P, find the minimum degree of connection between them.

Degree of connection is defined as:
• Directly connected users have degree 1
• Users connected through one intermediate friend have degree 2
• And so on...

If there's no path between the users, return -1.

Example 1:

Social Network Graph:
A ---- B ---- C | | | | | | D ---- E ---- F | | G
Input: connections = [["A","B"], ["A","D"], ["B","C"], ["B","E"], ["C","F"], ["D","E"], ["E","F"], ["E","G"]], S = "A", P = "G"
Output: 3
Explanation: Shortest path A → B → E → G (degree 3)

Example 2:

Simple Connection:
Alice ---- Bob ---- Charlie | | David
Input: connections = [["Alice","Bob"], ["Bob","Charlie"], ["Alice","David"]], S = "Alice", P = "Bob"
Output: 1
Explanation: Alice and Bob are directly connected

Example 3:

No Connection:
X ---- Y M ---- N | Z
Input: connections = [["X","Y"], ["Y","Z"], ["M","N"]], S = "X", P = "M"
Output: -1
Explanation: No path exists between X and M

Implement Unix Tail -N Command

Write a program that mimics the behavior of the Unix tail -n command. It should return the last N lines from a given input stream or file content.

Function Signature

List<String> tail(InputStream input, int N)

Example

Input:
Stream:
Line1
Line2
Line3
Line4
Line5

N = 3

Output:
["Line3", "Line4", "Line5"]

Constraints

  • 0 ≤ N ≤ 106
  • The number of lines in input can be up to 107
  • You must use constant memory relative to input size (i.e., avoid storing the entire file)

Design a Concurrent Task Scheduler

Design and implement a TaskScheduler class that allows scheduling of tasks to be executed after a specified delay. The scheduler should support concurrent scheduling and must execute tasks with correct timing and thread safety.

Implement the class:

TaskScheduler {
  TaskScheduler(int workerThreads)
  void schedule(Runnable task, long delayMillis)
  void shutdown()
}

Constraints:

  • Tasks must be executed as close as possible to their scheduled time.
  • Multiple tasks can be scheduled concurrently from different threads.
  • Should use thread-safe data structures and efficient locking.
  • Tasks with the same execution time may run in parallel (based on thread pool).

Example:


TaskScheduler scheduler = new TaskScheduler(2);
scheduler.schedule(() -> System.out.println("A"), 1000);
scheduler.schedule(() -> System.out.println("B"), 500);
scheduler.schedule(() -> System.out.println("C"), 1500);
scheduler.shutdown();

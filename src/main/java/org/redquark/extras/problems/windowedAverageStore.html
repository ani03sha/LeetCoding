Windowed Average Cache

Design a data structure that supports two operations:

  • add(int value, long timestamp): Adds a value to the cache with a given timestamp.
  • getAverage(long currentTime): Returns the average of all valid (non-expired) values.

A value is valid if it was added within a fixed windowSize duration (in milliseconds) before currentTime.

Example

Input:
window = new WindowedAverageCache(5000) // 5 second window
window.add(10, 1000)
window.add(20, 2000)
window.add(30, 6000)

window.getAverage(7000) ➞ 30.0  // Only (30, 6000) is valid
window.getAverage(2500) ➞ 15.0  // (10,1000) and (20,2000) are valid

Constraints

  • 0 ≤ value ≤ 106
  • 0 ≤ timestamp ≤ 1018
  • All timestamps are in non-decreasing order
  • All operations must be efficient (O(1) amortized preferred)

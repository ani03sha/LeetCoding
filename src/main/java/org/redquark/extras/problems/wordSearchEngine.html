Word Search Engine

Implement a basic word search engine. You are given a list of documents, each with an associated ID and text content. You must build a search engine that can:

  • Return document IDs that contain a given word (case-insensitive)
  • Follow-up: Extend the engine to support phrase searches (e.g., "hello world")

Example

Input:
documents = {
  1: "The quick brown fox",
  2: "Jumped over the lazy dog",
  3: "the quick dog is fast"
}

search("quick") ➞ [1, 3]
search("the quick") ➞ [1, 3]

Constraints

  • 1 ≤ number of documents ≤ 10⁴
  • 1 ≤ length of document text ≤ 10⁵
  • All input words are lowercase a-z with no punctuation

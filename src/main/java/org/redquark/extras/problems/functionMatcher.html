Function Signature Matcher

Given a list of function signatures and a list of input arguments, return all matching functions based on parameter compatibility.

Function Signature

List<String> findMatchingFunctions(List<String> functionSignatures, List<Class<?>> inputTypes)

Constraints

  • Function names are lowercase alphanumeric and may be overloaded.
  • Parameter types include classes like Integer, String, Number, Object.
  • Functions may include variadic parameters denoted as Type....
  • inputTypes contains Java class types (e.g., Integer.class).

Input

  • functionSignatures: List of strings representing function definitions.
  • inputTypes: List of Java class types representing the actual input types.

Output

A list of function signatures (as strings) that match the provided input types.

Example

Input:

functionSignatures = [
  "foo(Integer)",
  "foo(String, Integer)",
  "foo(Integer...)",
  "foo(String, Integer...)"
]

inputTypes = [Integer.class, Integer.class, Integer.class]

Output:

[
  "foo(Integer...)"
]

Follow-Up

  • Extend the solution to support primitive types like int, double.
  • Implement overload resolution that prioritizes exact matches over varargs.
  • Make the matching resilient to null inputs or ambiguous cases.

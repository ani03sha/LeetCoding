Cost Explorer

Difficulty: Medium

Problem Statement

You're part of a payments team building a Cost Explorer. Customers subscribe to various products, and your goal is to provide them with a cost estimate for the rest of the year.

The Cost Explorer must calculate:

  • Monthly cost report for the current year (including future months)
  • Total yearly cost estimate

Plans

[
  { BASIC, 9.99 },
  { STANDARD, 49.99 },
  { PREMIUM, 249.99 }
]

Function Signature

public class CostExplorer {
    public Map<Month, Double> getMonthlyReport(LocalDate startDate, Plan plan);
    public double getYearlyEstimate(LocalDate startDate, Plan plan);
  }

Example

Input:
startDate = 2025-06-15
plan = STANDARD

Output:
Monthly Report:
JUNE: 49.99
JULY: 49.99
...
DECEMBER: 49.99

Yearly Estimate:
349.93

Follow-up

Incorporate a trial period (in days). During the trial, the customer is not billed.

Constraints

  • All subscriptions start in the current year
  • Ignore leap years
  • Prices are monthly (flat rate)

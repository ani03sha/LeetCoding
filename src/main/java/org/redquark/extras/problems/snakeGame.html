Design Snake Game (Custom Version)

Description

Design the classic Snake game logic as described below.

The snake can move in four directions — up, down, left, or right — on a square board of fixed size. Initially, the snake has a length of 3 and grows by 1 unit every 5 moves.

The game ends when the snake hits itself.

Interface

interface SnakeGame {
    void moveSnake(String snakeDirection); // "U", "D", "L", "R"
    boolean isGameOver();
}

Rules

  • The board is square and has a fixed size.
  • The snake only moves when moveSnake is called.
  • The snake grows by 1 unit after every 5 moves.
  • The game ends when the snake hits its own body.

Constraints

  • 1 ≤ board size ≤ 100
  • Initial snake length is 3
  • The direction is always valid (within board boundaries)

Example

Input:
SnakeGame game = new SnakeGame(10);
game.moveSnake("R");
game.moveSnake("D");
...
game.isGameOver();

Output:
false

Fix and Optimize Quicksort

Given a buggy implementation of the Quicksort algorithm, fix any issues and ensure it sorts arrays correctly. Then, optimize the implementation by applying various pivot strategies to reduce the risk of degraded performance to O(n²).

Strategies to Apply

  • Randomized pivot selection
  • Median-of-three pivot
  • Dual-pivot Quicksort

Example

Input: nums = [3, 2, 1, 5, 4]
Output: [1, 2, 3, 4, 5]

Constraints

  • 1 ≤ nums.length ≤ 10⁵
  • -10⁹ ≤ nums[i] ≤ 10⁹

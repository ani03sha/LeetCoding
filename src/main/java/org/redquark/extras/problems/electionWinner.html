Election Winner

Difficulty: Easy

Description

You are given a list of votes, where each vote is represented by a candidate name (string).

Write a function to determine the winner — the candidate with the highest number of votes.

Example

Input: ["c1", "c2", "c1", "c2", "c1", "c2", "c3", "c4", "c4"]
Output: "c1"

Constraints

  • 1 ≤ number of votes ≤ 105
  • Each vote is a non-empty string of at most 10 characters

Follow-up

In case of a tie, suggest different tie-breaking strategies. Which is the most reasonable and why?

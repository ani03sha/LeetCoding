Teleporting Dice Game

Difficulty: Medium

Problem Statement

You're given a linear board with positions numbered from 0 to lastNumber. You begin at startPosition and can roll a die with face values from 1 to maxDieValue. After a roll, you move forward by the rolled number. If you land on a cell with a teleporter, you are instantly moved to its target cell.

If a die roll moves you past the last cell, you are teleported directly to lastNumber.

Input

  • int lastNumber — Final board position
  • int startPosition — Starting cell
  • int maxDieValue — Maximum value of the die
  • List<String> teleporters — Each in format "src,dest"

Part 1

Find all final positions reachable by rolling the die once (values 1 to maxDieValue), applying teleporters and edge-wrapping rule.

Example

Input:

lastNumber = 10
startPosition = 3
maxDieValue = 6
teleporters = ["3,1", "5,10", "8,2"]

Output:

[1, 6, 7, 10, 9]

Part 2 (Follow-up)

Can you reach lastNumber from startPosition using any number of die rolls?

Constraints

  • 0 ≤ startPosition ≤ lastNumber ≤ 104
  • 1 ≤ maxDieValue ≤ 20
  • teleporters are valid and do not create infinite loops
